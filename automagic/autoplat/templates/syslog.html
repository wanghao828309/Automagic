流量回放模式(Linux)

通过回放Syslog流进行发送syslog(UDP支持源地址伪装)|sendingdata.py 
usage:
sendingdata.py [-h] [-sip SRCIP]
[-dip DSTIP] [-p PROTOCOL]
[-dport DPORT] [-c COUNT]
[-f FILE] [-t THREAD]
[-s SPEED]
optional arguments:
-h, --help    show this help message
-sip SRCIP    src ip addr
-dip DSTIP    dst ip addr
-p PROTOCOL   protocol udp or tcp
-dport DPORT  send port
-c COUNT      packets count
-f FILE       packets file
-t THREAD     thread number
-s SPEED      send speed (s)

UDP Socket

通过UDP Socket发送syslog事件|udpsendingsyslog.py 
Usge:
python sendlog -f [filepath] -c [cycles]
-t [sleep time]
-d [Receiver Host] -p [Receiver port]
Details:
-C	[开启无限循环]
-f	[指派日志文件路径]
-c	[指定循环次数]
-t	[指定日志发送间隔时间 单位:秒]
-d	[指定日志接收IP地址]
-p	[指定日志接收端口（整数）]
-v	[查看工具版本]
-h	[查看帮助]

tcpsendingsyslog.py [-h] [-host HOST]
